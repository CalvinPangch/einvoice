1. open up the MokTHLibrary.WinTestAPI.exe.config
2. fill up those variable in the <appSettings>, save
3. run the MokTHLibrary.WinTestAPI.exe
4. In the Winform, fill in the invoice no, click Submit Invoice No. Wait, u will see the response in Json.
5. After submit, the Submission ID will auto fill up.
6. Click the Get Submit Status to get the document status.
7. Click the Get Document to get the document status (if the Status is Valid or Invalid).
8. Click the Get Document Detail to get the document detail status (if the Status is Valid or Invalid).
9. The Json file will store in the path u mentioned in the MokTHLibrary.WinTestAPI.exe.config.
